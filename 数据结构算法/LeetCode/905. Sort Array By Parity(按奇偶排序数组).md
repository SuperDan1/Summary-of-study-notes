# 905. Sort Array By Parity(按奇偶排序数组)

标签（空格分隔）： LeetCOde

---
给定一个非负整数数组 A，返回一个由 A 的所有偶数元素组成的数组，后面跟 A 的所有奇数元素。

你可以返回满足此条件的任何数组作为答案。
**示例**
> 输入：[3,1,2,4]
> 输出：[2,4,3,1]
> 输出 [4,2,3,1]，[2,4,1,3] 和 [4,2,1,3] 也会被接受。

解法一：

```C++
class Solution {
public:
    vector<int> sortArrayByParity(vector<int>& A) {
        int l = A.size();
        int temp;
        for (int i=0;i<l;i++)
        {
            if (A[i]%2==1)
            {
                temp = A[l-1];
                A[l-1] = A[i];
                A[i] = temp;
                l--;
                i--;
            }      
        }
        return A;
    }
};
```
